Spring Boot Kafka ML / Fraud AI Redis PostgreSQL
TxGuard — Real-Time Fraud Detection Engine

A production-grade fraud detection microservice that sits inline on the payment authorisation pipeline. Every card swipe, UPI push, or ATM withdrawal passes through a lightweight ML model (XGBoost via ONNX runtime) that scores the transaction in <10ms. Suspicious patterns — velocity checks, geo-anomalies, device fingerprint mismatches — trigger an automatic soft decline or step-up OTP challenge via Kafka events. Dashboards in Grafana show real-time fraud rates by network and channel.

Why it stands out: directly mirrors production payments infrastructure; combines ML inference with ISO-8583 message parsing — rare skill combo.

⚡ <10ms inference · 99.98% catch rate (simulated)

PayBot — LLM-Powered Banking Assistant

A Retrieval-Augmented Generation chatbot trained on RBI guidelines, UPI circulars, and card chargeback rules. Bank customers can ask natural-language questions about failed transactions, dispute timelines, and refund status. Backed by a Spring Boot service that fetches live transaction data and feeds it into context before each LLM call — so answers are always grounded in real account state, not hallucination.

Java Kafka Streams Anomaly AI Grafana
ClearFlow — Settlement Reconciliation AI

End-of-day settlement reconciliation is painful — files from card networks (Visa BASE II, RuPay) vs CBS ledger entries rarely match perfectly. ClearFlow automates the match/mismatch pipeline with a rule engine + anomaly detection model that flags suspicious unmatched entries, clusters them by error type, and generates a human-readable exception report — cutting manual reconciliation time from hours to minutes.
